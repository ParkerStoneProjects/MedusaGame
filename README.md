A small WIP top-down roguelike built in Unreal Engine 5.

## Features
- Petrify status effect system that freezes AI, pauses animations, and swaps materials
- Enemy AI built with AIControllers, movement logic, and condition-based behavior (stun/petrify)
- Off-screen enemy spawner using randomized angles/distances and navigation projection
- Wave-based spawning system using timers

## Tech Used
- Unreal Engine 5
- Blueprints
- Niagara
- Navigation System

## How to Open
1. Clone this repo
2. Open `MedusaGame.uproject` in Unreal Engine 5
3. Build & press Play
